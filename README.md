Shopping API Testing Project

Overview

This project is a local REST API testing lab created to practice API testing, troubleshooting, and SQL analysis skills.

The API is powered by JSON Server and contains four resources:

* Users
* Products
* Orders
* Payments

Objectives

* Practice REST API concepts
* Learn GET, POST, PUT, PATCH, and DELETE requests
* Test APIs using cURL, Postman, and Bruno
* Understand JSON request and response structures
* Work with filtering, sorting, and query parameters
* Build SQL queries using related datasets
* Practice Git and GitHub workflows

Technologies Used

* VS Code
* JSON Server
* Postman
* Bruno
* Git
* GitHub
* PostgreSQL

Sample Endpoints

GET /users

GET /products

GET /orders

GET /payments

POST /users

PUT /users/:id

PATCH /users/:id

DELETE /users/:id

Skills Demonstrated

* API Testing
* REST APIs
* JSON Processing
* Command Line Usage
* cURL
* SQL Fundamentals
* Troubleshooting
* Documentation
* Version Control
